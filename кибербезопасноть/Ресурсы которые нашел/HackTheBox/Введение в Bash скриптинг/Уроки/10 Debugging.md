Bash дает нам отличную возможность находить, отслеживать и исправлять ошибки в нашем коде. Термин `debugging`может иметь много разных значений. Тем не менее, [отладка Bash](https://tldp.org/LDP/Bash-Beginners-Guide/html/sect_02_03.html) - это процесс удаления ошибок (ошибок) из нашего кода. Отладка может быть выполнена различными способами. Например, мы можем использовать наш код для отладки для проверки опечаток, или мы можем использовать его для анализа кода, чтобы отслеживать их и определять, почему происходят конкретные ошибки.
Этот процесс также используется для поиска уязвимостей в программах. Например, мы можем попытаться вызвать ошибки с использованием различных типов ввода и отслеживать их обработку в процессоре через ассемблатор, что может обеспечить способ манипулирования обработкой этих ошибок, чтобы вставить наш собственный код и заставить систему выполнять его. Эта тема будет подробно освещаться и обсуждаться в других модулях. Bash позволяет нам отладить наш код, используя «`-x`" (`xtrace`) и "`-v`" варианты. Теперь давайте посмотрим пример с нашим `CIDR.sh`Сценарий.
#### CIDR.sh - Debugging
```shell
behzod789@htb[/htb]$ bash -x CIDR.sh

+ '[' 0 -eq 0 ']'
+ echo -e 'You need to specify the target domain.\n'
You need to specify the target domain.

+ echo -e Usage:
Usage:
+ echo -e '\tCIDR.sh <domain>'
    CIDR.sh <domain>
+ exit 1
```
Здесь Bash показывает нам, какая именно функция или команда выполнялись с какими значениями. Это обозначается знаком плюс (`+`) в начале линии. Если мы хотим увидеть весь код для конкретной функции, мы можем установить «`-v`" опция, которая отображает выход более подробно.
#### CIDR.sh - Verbose Debugging
```shell
behzod789@htb[/htb]$ bash -x -v CIDR.sh

#!/bin/bash

# Check for given argument
if [ $# -eq 0 ]
then
    echo -e "You need to specify the target domain.\n"
    echo -e "Usage:"
    echo -e "\t$0 <domain>"
    exit 1
else
    domain=$1
fi
+ '[' 0 -eq 0 ']'
+ echo -e 'You need to specify the target domain.\n'
You need to specify the target domain.

+ echo -e Usage:
Usage:
+ echo -e '\tCIDR.sh <domain>'
    CIDR.sh <domain>
+ exit 1
```
По сравнению с обычной отладкой, мы видим весь раздел кода, который был обработан до сих пор, а затем отдельные шаги, которые были предприняты.
