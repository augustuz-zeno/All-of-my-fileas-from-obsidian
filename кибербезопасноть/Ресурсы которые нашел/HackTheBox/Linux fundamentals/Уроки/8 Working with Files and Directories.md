## Linux File Management: Конспект

**Ключевые моменты:**

*   **Разница с Windows:** В Linux файловый менеджер – это терминал, обеспечивающий более быстрый и эффективный доступ и редактирование файлов, чем графические интерфейсы.
*   **Терминал и команды:** Терминал позволяет управлять файлами через команды, включая:
    *   **Создание файлов/директорий:**
        *   `touch <name>` – создает пустой файл.
        *   `mkdir <name>` – создает новую директорию.
        *   `mkdir -p <path>` – создает директорию и все ее родительские директории.
    *   **Переименование файлов/директорий:** `mv <file/directory> <new name>`
    *   **Копирование файлов:** `cp <source> <destination>`
    *   **Удаление файлов/директорий:** (Рассмотрено в разделе "Optional Exercise")
    *   **Redirection:**  Перенаправление ввода/вывода между командами и файлами для повышения эффективности.
    *   **Текстовые редакторы:** `vim`, `nano` (для интерактивного редактирования файлов).
*   **Регулярные выражения (regex):** Терминал позволяет использовать регулярные выражения для селективного изменения файлов.
*   **Автоматизация:** Возможность запускать несколько команд одновременно и перенаправлять вывод для автоматизации задач.

**Структура основных команд:**

| Команда   | Описание                               | Пример                               |
| :-------- | :------------------------------------- | :----------------------------------- |
| `touch`   | Создает пустой файл.                   | `touch info.txt`                      |
| `mkdir`   | Создает директорию.                    | `mkdir Storage`                       |
| `mkdir -p`| Создает директорию и все родительские. | `mkdir -p Storage/local/user/documents`|
| `mv`      | Перемещает/переименовывает файлы/директории.| `mv info.txt information.txt`         |
| `cp`      | Копирует файлы.                        | `cp Storage/readme.txt Storage/local/` |

**Инструменты:**

*   `tree`:  Визуальное отображение структуры директорий.
*   Терминал: Основной инструмент для работы с файлами в Linux.

**Рекомендации:**

*   Изучите и практикуйте основные команды.
*   Не бойтесь экспериментировать и использовать онлайн-ресурсы для поиска решений.
