[[0 HTB Linux fundamentals]]
## Конспект текста "HTB Linux Fundamentals"

**I. Введение и Общие Принципы**

*   **Навигация - Основа:**  В Linux навигация аналогична работе с мышью в Windows. Она необходима для перемещения по системе, работы с каталогами и файлами.
*   **Эксперименты - Лучший Способ Обучения:** Рекомендуется начинать с экспериментов, чтобы лучше понять принципы работы Linux.
*   **Резервное Копирование:** При работе на локальном VM настоятельно рекомендуется создавать snapshot для предотвращения потери данных в случае неисправности системы.

**II. Базовые Команды Навигации**

*   **`pwd` (Print Working Directory):**  Команда для определения текущего каталога, в котором находится пользователь.  Пример: `cry0l1t3@htb[~]$ pwd /home/cry0l1t3`
*   **`ls` (List):** Команда для просмотра содержимого каталога.
    *   Без опций отображает имена файлов и каталогов.
    *   С опцией `-l` отображает подробную информацию о файлах и каталогах (см. таблицу ниже).
*   **`ls -la`:** Опция `-la`  показывает все файлы, включая скрытые, начинающиеся с точки (например, `.bashrc`).

| **Column Content** | **Description**                                                                  |
| ------------------ | -------------------------------------------------------------------------------- |
| `drwxr-xr-x`       | Тип и права доступа (d - directory, - - - - - - permissions)                            |
| `2`                | Количество жестких ссылок (hard links) к файлу/каталогу                              |
| `cry0l1t3`         | Владелец файла/каталога                                                         |
| `htbacademy`       | Группа владельца файла/каталога                                                   |
| `4096`             | Размер файла (в байтах) или количество блоков, используемых для хранения информации о каталоге |
| `Nov 13 17:37`     | Дата и время последнего изменения файла/каталога                                     |
| `Desktop`          | Имя каталога                                                                    |

**III. Дополнительные Команды Навигации**

*   **`cd` (Change Directory):** Команда для перехода в другой каталог.
    *   Полный путь: `cd /dev/shm`
    *   Переход в предыдущий каталог: `cd -`
    *   Автодополнение:  При вводе `cd /dev/s` и нажатии `[TAB]` дважды, shell предложит список каталогов, начинающихся с "s" в каталоге `/dev/`.
*   **`cd ..`:** Команда для перехода в родительский каталог.
*   **`clear`:**  Команда для очистки экрана терминала.
*   **`[Ctrl] + [L]`:** Альтернативный способ очистки экрана.
*   **`↑` / `↓` (стрелки вверх/вниз):**  Навигация по истории команд.
*   **`[Ctrl] + [R]`:** Поиск в истории команд.

**IV. Другие полезные команды и концепции**

*   **Определение текущего каталога:** `pwd`
*   **Просмотр содержимого каталога:** `ls`
*   **Указание пути к каталогу при использовании `ls`:** `ls -l /var/`
*   **Автодополнение:**  Использование `[TAB]` для автоматического завершения команд и имен файлов.
*   **Скрытые файлы:**  Файлы, имена которых начинаются с точки (`.`).
*   **Файловые дескрипторы:** Концепция, которая не рассматривается в данном тексте, но является важной для понимания работы системы.
