# Bash Script: IP Sweep
Простой bash-скрипт для поиска активных устройств в сети.
Он перебирает IP-адреса и проверяет, какие отвечают на ping.
# Полный скрипт
```bash
#!/bin/bash

if [ "$1" == "" ]
then
echo "You forgot an IP address!"
echo "Syntax: ./ipsweep.sh 192.168.1"

else
for ip in `seq 1 254`; do
ping -c 1 $1.$ip | grep "64 bytes" | cut -d " " -f 4 | tr -d ":" &
done
fi
```
# Shebang
```bash
#!/bin/bash
```
Сообщает системе, что скрипт нужно выполнять через **Bash**.
# Проверка аргумента
```bash
if [ "$1" == "" ]
```
`$1` — <mark style="background:#ff4d4f">первый аргумент скрипта.</mark>
Пример запуска:
```bash
./ipsweep.sh 192.168.1
```
<mark style="background:#ff4d4f">Если аргумент не передан — выводится ошибка.</mark>
# Цикл перебора IP
```bash
for ip in `seq 1 254`
```
`seq 1 254` создаёт последовательность чисел:
```
1 2 3 ... 254
```
В результате проверяются IP:
```
192.168.1.1
192.168.1.2
192.168.1.3
...
192.168.1.254
```
# Проверка хоста
```bash
ping -c 1 $1.$ip
```
`-c 1` — отправить один ping-пакет.
# Фильтрация ответа
```bash
grep "64 bytes"
```
Показывает только успешные ответы.
# Извлечение IP
```bash
cut -d " " -f 4
```
Берёт 4-е поле строки (IP-адрес).
# Удаление двоеточия
```bash
tr -d ":"
```
Удаляет символ `:`.
# Фоновое выполнение
```bash
&
```
Запускает команды параллельно для ускорения сканирования.
# Пример запуска
```bash
chmod +x ipsweep.sh
./ipsweep.sh 192.168.1
```
Возможный результат:
```
192.168.1.1
192.168.1.5
192.168.1.10
192.168.1.23
```
Это активные устройства в сети.
Важно понимать: этот скрипт — **очень базовая версия сетевого сканера**. Профессионально такие вещи делают инструментами вроде:
- `nmap`
- `arp-scan`
- `netdiscover`
Но подобный скрипт часто используют на первых уроках, чтобы показать:
- циклы в bash
- обработку текста (`grep`, `cut`, `tr`)
- работу с сетью.
