# Команды для обновления системы, установки программ и скачивания проектов из GitHub.
# sudo
`sudo` выполняет команду с правами администратора (root).
Многие операции управления системой требуют этих прав.
Пример:
```bash
sudo command
```
# APT — пакетный менеджер
APT используется для управления программами в Debian-based системах:
- Ubuntu
- Kali Linux
- Debian
- Linux Mint
Он позволяет:
- устанавливать программы
- обновлять систему
- удалять пакеты
# Обновление системы
```bash
sudo apt update && sudo apt upgrade
```
Команда состоит из двух частей.
## sudo apt update
```bash
sudo apt update
```
Что делает:
- обновляет список доступных пакетов
- получает информацию из репозиториев
Важно:  
она **не обновляет программы**, а только обновляет список обновлений.
## sudo apt upgrade
```bash
sudo apt upgrade
```
Что делает:
- устанавливает новые версии уже установленных пакетов
- обновляет систему
## Оператор &&
```bash
command1 && command2
```
Означает:
выполнить вторую команду **только если первая прошла успешно**.
Поэтому:
```bash
sudo apt update && sudo apt upgrade
```
сначала обновляет список пакетов, затем обновляет систему.
# Установка программ
## apt install
Команда для установки нового пакета.
```bash
sudo apt install package_name
```
## Пример установки cron
```bash
sudo apt install cron-daemon-common
```
### Что такое cron
`cron` — это планировщик задач в Linux.
Он позволяет автоматически запускать команды по расписанию.
Примеры использования:
- резервные копии
- автоматические обновления
- очистка логов
- запуск скриптов
# Git
Git — это система контроля версий.
Она используется для:
- хранения кода
- совместной разработки
- отслеживания изменений
GitHub — один из самых популярных сервисов для Git-репозиториев.
# Клонирование репозитория
```bash
git clone URL
```
Эта команда скачивает проект из удалённого репозитория.
## Пример
```bash
sudo git clone https://github.com/Dewalt-arch/pimpmykali.git
```
Что происходит:
- Git подключается к GitHub
- скачивает весь проект
- создаёт папку `pimpmykali`
- загружает все файлы и историю изменений
# Что такое pimpmykali
`pimpmykali` — это скрипт для настройки Kali Linux.
Он может:
- обновлять систему
- устанавливать инструменты
- настраивать окружение
# Типичный workflow
Обновить систему:
```bash
sudo apt update && sudo apt upgrade
```
Установить программу:
```bash
sudo apt install program_name
```
Скачать проект с GitHub:
```bash
git clone repository_url
```
