# Базовые команды для создания и редактирования файлов в Linux.
# Перенаправление вывода (Redirection)

В Linux результат выполнения команды можно направить:
- на экран (по умолчанию)
- в файл
Для этого используются операторы:
- `>` — перезаписать файл
- `>>` — добавить в конец файла
# echo
Команда `echo` выводит текст в терминал.
### Пример
```bash
echo "hello"
```
Вывод:
```
hello
```
# Запись / Перезапись в файл
```bash
echo "hello" > hey.txt
```
Что происходит:
- создается файл `hey.txt`
- в него записывается текст `hello`
- если файл уже существует — его содержимое **полностью удаляется**
### Результат файла
```
hello
```
## Добавление текста в файл

```bash
echo "hello again" >> hey.txt
```
Что происходит:
- текст добавляется **в конец файла**
- если файла нет — он будет создан
### Результат файла
```
hello
hello again
```
# touch
Команда для создания пустого файла.
```bash
touch newfile.txt
```
Что происходит:
- создаётся пустой файл `newfile.txt`
Если файл уже существует:
- изменяется его **timestamp** (время последнего изменения)
# Редактирование файлов
## nano
`nano` — простой терминальный текстовый редактор.
```bash
nano newfile.txt
```
Что происходит:
- открывается редактор
- можно писать или изменять текст файла
Основные горячие клавиши:

| Команда   | Клавиши   |
| --------- | --------- |
| сохранить | CTRL + O  |
| выйти     | CTRL + X  |
| поиск     | CTRL + W  |

## mousepad
`mousepad` — графический текстовый редактор (GUI).
```bash
mousepad newfile.txt
```
Что происходит:
- файл открывается в графическом окне
- редактирование происходит как в обычном текстовом редакторе
Используется в окружениях XFCE.
# Мини-практика
Создать файл:
```bash
touch notes.txt
```
Записать текст:
```bash
echo "first line" > notes.txt
```
Добавить текст:
```bash
echo "second line" >> notes.txt
```
Открыть файл:
```bash
nano notes.txt
```
