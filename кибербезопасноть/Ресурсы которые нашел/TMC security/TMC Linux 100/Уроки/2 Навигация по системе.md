первая команда pwd - который я уже разбирал в [[Lesson 2 18.02.2026]]. Далее тоже почти те же команды команды, есть и новые, только продемонстрировал их с некоторыми параметрами

| cd ..                                      | Перемещает на одну директорию назад                                                                                                      |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- |
| ls <название_директории>                   | Можно просмотреть список файлов и папков в одной какой либо директории, находясь в другой![[{C6E52DB7-125D-4300-A896-99E145E739BD}.png]] |
| mkdir <название_директории>                | С данной командой можно создать директорию.                                                                                              |
| rmdir <название_директории>                | Делает точно наоборот, а именно удаляет директорию                                                                                       |
| <команда> --help                           | Работает так же как в случае man, только выводит текст в консоль                                                                         |
| echo 'Текст'                               | Выводит написанный текст в консоли                                                                                                       |
| echo 'Текст' > <название>.<расширение>     | В данном случае echo сохраняет текст, в созданном файле, для примера test.txt<br>![[{8BCF6776-378E-44F9-A3DE-1FB974F25138}.png]]         |
| cat <название_файла>                       | Выводит содержимое файла в терминал.                                                                                                     |
| cp <название-файла> <путь-куда-копировать> | Копирует файл в нужную директорию, пример ниже<br>![[{CD8DA373-CB3E-4F3F-9B10-39DE7B91ADA4}.png]]                                        |
| rm <название_файла>                        | Удаляет файл, пример:<br>![[{C56DF193-6772-4949-9EA9-E33595AAE346}.png]]                                                                 |
| rm  <путь-где-он> <название-файла>         | Удаляет файл в одной директории даже находясь в другой, пример:<br>![[{9C3B6F4B-1714-4894-B05D-D4033E655751}.png]]                       |
| mv <название-файла> <путь-куда-нужно>      | Перемещает файл туда куда нужно.                                                                                                         |
| locate <название-файла>                    | Находит файл который нужен, но возможно потребуется **updatedb**.                                                                        |
| update                                     | Для выполнения команды нужны будут root права, и оно обновляет базу данных, чтобы в системе обновились недавние действия.                |
| passwd                                     | Позволяет поменять пароль.                                                                                                               |
