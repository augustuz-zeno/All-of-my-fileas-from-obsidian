## Командная строка (CMD / PowerShell / WMIC / WSL)

- **`cd`** — смена директории (аналог в Linux).
    
- **`type`** — чтение содержимого файлов (аналог **`cat`** в Linux).
    
- **`dir`** — вывод списка файлов и папок (аналог **`ls`** в Linux).
    
- **`dir /a`** — вывод списка всех файлов и папок (включая скрытые).
    
- **`mklink`** — создание символической ссылки (soft link).
    
- **`mklink /h`** — создание жесткой ссылки (hard link).
    
- **`find` / `findstr`** — поиск текста в файлах (аналог **`grep`** в Linux).
    
- **`cls`** — очистка экрана (аналог **`clear`** в Linux).
    
- **`whoami`** — вывод имени текущего пользователя.
    
- **`help [команда]`** или **`[команда] /?`** — справка о конкретной команде.
## Управление пользователями (`net user`)
- **`net user`** — список всех пользователей системы.
    
- **`net localgroup`** — список локальных групп.
    
- **`net user qwerty qwerty /add`** — создание нового пользователя (логин: `qwerty`, пароль: `qwerty`).
    
- **`net user qwerty /delete`** — удаление пользователя `qwerty`.
## Сетевые команды
- **`ipconfig`** — вывод сетевых настроек (IP-адреса, маски и т.д.).
    
- **`ipconfig /release`** — освобождение текущего IP-адреса.
    
- **`ipconfig /renew`** — обновление/получение нового IP-адреса.
    
- **`netstat`** — отображение активных сетевых соединений и обмена пакетами.
## Переменные окружения
- **Переменная** — область памяти для временного хранения определенного значения.
    
- **`set u12=cyber`** — создание временной переменной окружения (действует только в текущей сессии).
    
- **`setx u11 "hello world" /m`** — создание постоянной системной переменной окружения (для всей системы).
## Общие папки и сетевые диски (Windows / SMB)
- **`net share`** — просмотр всех общих (расшаренных) ресурсов на ПК.
    
- **`net share u13=C:\Users\predator\Downloads /grant:Everyone,Full`** — открытие полного сетевого доступа к папке под именем `u13` для всех.
    
- **`net use Z: \\127.0.0.1\u13`** — подключение удаленной папки `u13` как сетевого диска `Z:`.
    
- **`fsutil fsinfo drives`** — вывод списка всех дисков в системе (C:, D: и т.д.).
    
- **`smbclient -L \\ip -U username`** — просмотр доступных общих ресурсов на удаленном IP через Linux (SMB).
    
- **`smbclient //127.0.0.1/u13 -U username`** — подключение к общему ресурсу по протоколу SMB.
    
- **`psexec`** — утилита для удаленного выполнения команд.
## Безопасность и хэши
- **NTLM (New Technology LAN Manager)** — тип хэша, используемый в Windows для хранения паролей.
    
- **`impacket-secretsdump -sam sam -system system LOCAL`** — извлечение хэшей паролей из файлов SAM и SYSTEM на локальной машине.
