## Конспект: Передача данных и работа с HTTP/DNS
---
### 1. Копирование файлов через SCP
Команда `scp` (Secure Copy) используется для безопасной передачи файлов между узлами в сети через SSH.
- **Загрузка на удаленный сервер:**
```Bash
scp -P 1122 /путь/к/локальному/файлу user@ip:/путь/на/сервере
```
- **Скачивание с удаленного сервера:**
```Bash
scp -P 1122 user@ip:/путь/к/удаленному/файлу/test.txt ./локальное_имя.txt
```
*   `-P 1122` — указание специфического порта SSH.
---
### 2. Утилита Wget
Предназначена для скачивания файлов и веб-страниц напрямую из сети.
*   **Базовое скачивание:**
    `wget [https://example.com/file.pdf](https://example.com/file.pdf)` — загружает файл в текущую директорию.
*   **Ключевые флаги:**
    *   `-O` — сохранить файл под другим именем.
    *   `-o` — сохранить логи процесса загрузки в отдельный файл.
---
### 3. Методы HTTP
Основные типы запросов, которые клиент отправляет серверу:

| Метод      | Описание                                                     |
| :--------- | :----------------------------------------------------------- |
| **GET**    | Получение данных с сервера.                                  |
| **POST**   | Отправка новых данных на сервер (например, форм или файлов). |
| **PUT**    | Обновление существующих данных на сервере.                   |
| **DELETE** | Удаление указанного ресурса с сервера.                       |

---
### 4. Утилита Curl
Аналог `wget`, но с расширенным функционалом. Позволяет гибко работать с HTTP-заголовками и изменять методы запросов для тестирования взаимодействия с сервером.

---
### 5. Работа с DNS
Инструменты для получения информации о доменных именах и IP-адресах:
*   **dig** — мощная утилита для опроса DNS-серверов и получения подробных записей.
*   **host** — простая утилита для быстрого поиска соответствия имени и IP.
*   **nslookup** — классический инструмент для интерактивной проверки DNS-записей.
