# 1. Введение
В Linux существует несколько мощных инструментов для поиска и обработки текста:

| Команда | Назначение                |
| ------- | ------------------------- |
| grep    | поиск строк               |
| sed     | изменение и замена текста |
| cut     | извлечение колонок        |
| awk     | анализ и обработка текста |

Эти команды часто используются вместе через **pipe `|`**.
Пример:
```bash
cat file.txt | grep error | sort
```
Что происходит:
1. читается файл
2. ищутся строки `error`
3. результат сортируется
# 2. Regex (Regular Expressions)
**Regex** — это шаблоны для поиска текста.
Примеры:

| Шаблон | Значение               |
| ------ | ---------------------- |
| .      | любой символ           |
| *      | повторение             |
| ^      | начало строки          |
| $      | конец строки           |
| [abc]  | любой символ из списка |

Пример:
```bash
grep "^root" /etc/passwd
```
Найдёт строки, которые начинаются с `root`.

# 3. grep — Поиск текста
`grep` используется для поиска строк по шаблону.
## Синтаксис
```bash
grep [опции] "pattern" файл
```
## Основные опции

| Опция | Значение                      |
| ----- | ----------------------------- |
| -i    | игнорировать регистр          |
| -r    | рекурсивный поиск             |
| -v    | показать несовпадающие строки |
| -n    | показать номер строки         |

## Примеры
Поиск файлов:
```bash
ls -la /usr/bin | grep zip
```
Игнорирование регистра:
```bash
grep -i password file.txt
```
Исключение строк:
```bash
grep -v nologin /etc/passwd
```

# 4. sed — Stream Editor
`sed` используется для автоматического изменения текста.
## Замена текста
```bash
sed 's/old/new/'
```
Пример:
```bash
echo "I need to try hard" | sed 's/hard/harder/'
```
Результат:
```
I need to try harder
```
## Глобальная замена
```bash
sed 's/bin/HTB/g'
```
`g` означает заменить **все совпадения**.

---
# 5. cut — Извлечение полей
`cut` извлекает части строки.
## Основные опции

| Опция | Значение    |
| ----- | ----------- |
| -d    | разделитель |
| -f    | номер поля  |
## Пример
```bash
echo "I hack binaries,web apps,mobile apps" | cut -f2 -d ","
```
Результат:
```
web apps
```
## Пример с пользователями
```bash
cut -d ":" -f1 /etc/passwd
```
Покажет имена пользователей.

---
# 6. awk — Анализ текста
`awk` — мощный инструмент анализа текста.
## Основной синтаксис
```bash
awk '{print $1}'
```
Переменные:

| Переменная | Значение          |
| ---------- | ----------------- |
| $1         | первый столбец    |
| $2         | второй столбец    |
| $NF        | последний столбец |

## Пример
```bash
echo "hello::there::friend" | awk -F "::" '{print $1, $3}'
```
Результат:
```
hello friend
```
## Почему awk мощнее cut

| cut                     | awk                |
| ----------------------- | ------------------ |
| один символ-разделитель | несколько символов |
| только извлечение       | анализ и логика    |

# 7. Сравнение файлов
## comm
Сравнивает **два отсортированных файла**.
Вывод содержит 3 колонки:

| Колонка | Значение                |
| ------- | ----------------------- |
| 1       | строки только в файле 1 |
| 2       | строки только в файле 2 |
| 3       | строки в обоих файлах   |

Показать только общие строки:
```bash
comm -12 file1 file2
```

---
## diff

Показывает различия между файлами.
Опции:

| Опция | Значение       |
| ----- | -------------- |
| -c    | context format |
| -u    | unified format |

Обозначения:

| Символ | Значение                   |
| ------ | -------------------------- |
| -      | строка есть в первом файле |
| +      | строка есть во втором      |

# 8. nano — Редактор текста
`nano` — простой терминальный редактор.
## Открыть файл
```bash
nano file.txt
```
## Основные сочетания клавиш

| Клавиши          | Действие             |
| ---------------- | -------------------- |
| Ctrl + O         | сохранить            |
| Ctrl + X         | выйти                |
| Ctrl + W         | поиск                |
| Ctrl + W → Enter | следующее совпадение |

# 9. Фильтрация данных (HTB)

Команды для просмотра и обработки больших файлов.

---
## more
Просмотр файла по страницам.
```bash
cat /etc/passwd | more
```
Навигация:

| Клавиша | Действие           |
| ------- | ------------------ |
| Space   | следующая страница |
| Enter   | следующая строка   |
| Q       | выход              |

---

# Pipeline — главная идея Linux
Команды можно соединять:
```bash
command | command | command
```
Пример:
```bash
cat /etc/passwd | grep bash | cut -d ":" -f1
```
Что происходит:
1. читается файл
2. ищутся пользователи с bash
3. выводится только имя пользователя
