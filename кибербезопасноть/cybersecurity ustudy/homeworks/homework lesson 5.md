Что такое команды vim, nano, touch, mkdir и для чего они используются?

| Название команды | Функция                                                                                                                                                                                                                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `vim`            | Консольный текстовый редактор. Это мощный инструмент с модальным управлением (режим вставки, командный режим и т.д.). Используется для редактирования конфигурационных файлов, кода, скриптов и любых текстовых данных прямо в терминале. Часто установлен по умолчанию на серверах. |
| `nano`           | Более простой текстовый редактор в терминале. Управляется через подсказки внизу экрана (Ctrl+O — сохранить, Ctrl+X — выйти). Подходит для базового редактирования файлов без необходимости изучать сложные команды.                                                                  |
| `touch`          | Команда для создания пустого файла или изменения времени его последней модификации. Если файл не существует, он создаётся. Если существует — обновляется временная метка.                                                                                                            |
| `mkdir`          | создаёт новый каталог (директорию). Например, `mkdir project` создаст папку project. С ключом `-p` можно создавать вложенные каталоги сразу, например `mkdir -p a/b/c`                                                                                                               |
в целом эти команды уже до этого тоже применял, а `mkdir` уже был разобран в [[2 Навигация по системе]].
