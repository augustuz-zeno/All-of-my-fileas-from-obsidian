### Вопросы и ответы по сетевым технологиям

| Вопрос                                                              | Ответ                                                                                                                                                                                                            |
| ------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. В чем разница между WLAN и LAN?**                              | **LAN** (Local Area Network) — это проводная локальная сеть. **WLAN** (Wireless LAN) — это беспроводная локальная сеть, использующая Wi-Fi.                                                                      |
| **2. Приведите примеры сети PAN.**                                  | **PAN** (Personal Area Network) — персональная сеть. Примеры: соединение смартфона с Bluetooth-наушниками или смарт-часами.                                                                                      |
| **3. Напишите 3 типа VPN.**                                         | 1. Remote Access (удаленный доступ). 2. Site-to-Site (между офисами). 3. Client-to-Provider (используется обычными пользователями для обхода блокировок).                                                        |
| **4. Что вы понимаете под термином Network?**                       | Это группа систем (компьютеров, серверов и др.), соединенных для обмена данными и ресурсами.                                                                                                                     |
| **5. Что такое HTTP, HTTPS, FTP, SMTP, DNS, UDP, TCP?**             | Это сетевые протоколы: **HTTP/S** — просмотр сайтов; **FTP** — передача файлов; **SMTP** — почта; **DNS** — преобразование имен сайтов в IP; **TCP/UDP** — способы передачи данных (с гарантией доставки и без). |
| **6. Что такое конечное устройство (End device/Host)?**             | Устройство, которое является источником или пунктом назначения данных в сети. Важно, так как именно на нем работает пользователь.                                                                                |
| **7. Три примера конечных устройств.**                              | 1. **ПК**: для работы с программами. 2. **Смартфон**: мобильный доступ к сети. 3. **IP-принтер**: печать документов по сети.                                                                                     |
| **8. Какую функцию выполняют промежуточные устройства?**            | Обеспечивают связь между конечными устройствами, управляют потоками данных и восстанавливают сигналы.                                                                                                            |
| **9. Четыре типа промежуточных устройств.**                         | 1. **Router**: соединяет разные сети. 2. **Switch**: соединяет устройства в одной сети. 3. **Firewall**: безопасность. 4. **WAP**: точка беспроводного доступа.                                                  |
| **10. Что такое NIC и его функция?**                                | **Сетевая карта** (Network Interface Card) — аппаратный компонент для подключения компьютера к компьютерной сети.                                                                                                |
| **11. Почему важен MAC-адрес и его уровень в OSI?**                 | Уникальный физический идентификатор устройства. Работает на 2-м уровне (**Data Link**).                                                                                                                          |
| **12. Что такое Router и как он определяет путь?**                  | Устройство для пересылки пакетов между сетями. Определяет путь на основе **таблицы маршрутизации** и IP-адреса назначения.                                                                                       |
| **13. Разница между Switch и Hub в передаче данных.**               | **Hub** рассылает данные всем портам (широковещание). **Switch** передает данные только конкретному получателю по его MAC-адресу.                                                                                |
| **14. Какой протокол сопоставляет IP с MAC-адресом?**               | **ARP** (Address Resolution Protocol).                                                                                                                                                                           |
| **15. Какая версия IP использует 128-битные адреса?**               | **IPv6**.                                                                                                                                                                                                        |
| **16. На каком уровне OSI работают порты?**                         | На 4-м уровне (**Transport** — Транспортный).                                                                                                                                                                    |
| **17. Какой номер порта закреплен за HTTP?**                        | Порт **80**.                                                                                                                                                                                                     |
| **18. Опишите процесс DORA в DHCP.**                                | 1. **D**iscovery (поиск сервера). 2. **O**ffer (предложение IP). 3. **R**equest (запрос IP пользователем). 4. **A**cknowledge (подтверждение аренды).                                                            |
| **19. Разница между Static NAT, Dynamic NAT и PAT.**                | **Static**: 1 внутренний IP = 1 внешний IP. **Dynamic**: группа внутренних на группу внешних. **PAT**: много внутренних на 1 внешний IP (через разные порты).                                                    |
| **20. Что такое иерархия DNS?**                                     | Структура системы доменных имен: Root (корень) → TLD (.com, .ru) → Домен 2-го уровня (https://www.google.com/search?q=google.com).                                                                               |
| **21. Опишите процесс DNS Resolution.**                             | Клиент запрашивает имя -> Рекурсивный резолвер ищет ответ у Root, TLD и Authoritative серверов -> IP возвращается клиенту.                                                                                       |
| **22. Что такое DHCP reservation?**                                 | Резервирование конкретного IP-адреса за определенным устройством (по MAC-адресу), чтобы оно всегда получало один и тот же IP.                                                                                    |
| **23. Что такое таблица NAT?**                                      | База данных соответствий между внутренними (локальными) и внешними (публичными) адресами и портами.                                                                                                              |
| **24. Какие волны используют беспроводные сети?**                   | Радиоволны (обычно частоты 2.4 ГГц и 5 ГГц).                                                                                                                                                                     |
| **25. Какое устройство в доме объединяет функции роутера и Wi-Fi?** | **Беспроводной маршрутизатор** (Wireless Router).                                                                                                                                                                |
| **26. Что использует Mobile hotspot для подключения?**              | Мобильные сотовые данные (3G/4G/5G/LTE).                                                                                                                                                                         |
| **27. Какой диапазон частот лучше проходит сквозь стены?**          | **2.4 ГГц** (но он более подвержен помехам по сравнению с 5 ГГц).                                                                                                                                                |
