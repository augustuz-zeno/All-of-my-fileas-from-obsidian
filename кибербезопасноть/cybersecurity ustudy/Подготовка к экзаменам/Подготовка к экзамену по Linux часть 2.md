## 1. Управление пользователями и группами

### Ключевые файлы системы

|Файл|Доступ|Содержание|Формат записи|
|---|---|---|---|
|`/etc/passwd`|Всем (r)|Данные аккаунтов|`user:x:UID:GID:comment:home:shell`|
|`/etc/shadow`|Только Root|Хэши и политики|`user:hash:last:min:max:warn:inact:exp`|
|`/etc/group`|Всем (r)|Данные групп|`group:x:GID:users`|


**Детали по `/etc/shadow`:**

- **Идентификаторы хэшей:** `$6$` (SHA-512), `$5$` (SHA-256), `$1$` (MD5).
    
- **Спец-символы:** `!` — заблокирован, `*` — вход запрещен.
    
- **UID 0** — всегда принадлежит `root` (полный доступ).
    

### Администрирование

- **Блокировка пароля:** `passwd -l username` или `usermod -L username` (добавляет `!` в shadow).
    
- **Отключение оболочки:** `usermod -s /sbin/nologin username`.
    
- **Контроль сроков (Expire):** `chage -E 2026-12-31 username`. Проверка: `chage -l username`.
    

### Привилегии (sudo & su)

- `sudo -l` — проверить разрешенные команды в `/etc/sudoers`.
    
- `sudo -i` — интерактивная оболочка root.
    
- `su -l user -c "cmd"` — выполнить команду от имени другого пользователя.

## 2. Поиск файлов и программ

|Команда|Назначение|Плюсы|
|---|---|---|
|`which`|Найти путь к исполняемому файлу (python, curl)|Скорость|
|`locate`|Поиск по всей системе через БД (`sudo updatedb`)|Скорость|
|`find`|Поиск в реальном времени по сложным критериям|Гибкость фильтров|


**Сложный пример `find`:** Найти файлы `.conf` в `/`, принадлежащие `root`, размером более 20КБ, измененные после 3 марта 2020, и сразу выполнить `ls -al`:

Bash

```
find / -type f -name "*.conf" -user root -size +20k -newermt 2020-03-03 -exec ls -al {} \; 2>/dev/null
```

## 3. Обработка и фильтрация текста

### Основные утилиты

- **`grep` (Поиск):** * `-i` (игнор регистра), `-v` (инверсия), `-r` (рекурсивно).
    
    - Пример: `grep "^root" /etc/passwd` (строки, начинающиеся на root).
        
- **`sed` (Замена):** * Пример: `sed 's/bin/HTB/g'` (заменить все bin на HTB).
    
- **`cut` (Извлечение):** * Пример: `cut -d ":" -f1 /etc/passwd` (вырезать первую колонку, разделитель `:`).
    
- **`awk` (Анализ):** * Пример: `awk -F "::" '{print $1, $NF}'` (вывести первую и последнюю колонку).
    

### Сравнение файлов

- `comm -12 f1 f2` — вывести строки, общие для двух отсортированных файлов.
    
- `diff -u f1 f2` — показать различия в формате unified.

## 4. Просмотр файлов и статистика

- `less` — чтение с прокруткой и поиском (`/слово`).
    
- `head` / `tail -n 20` — первые/последние 20 строк.
    
- `sort -u` — сортировка с удалением дубликатов.
    
- `wc -l` — подсчет количества строк.
    
- `tr ":" " "` — замена символов (двоеточия на пробелы).
    
- `column -t -s ":"` — форматирование вывода в таблицу.

## 5. Права доступа и Ownership

**Базовые права (Octal):** `r` (4) — чтение, `w` (2) — запись, `x` (1) — выполнение. Пример: `-rwxr-x---` (Владелец: 7, Группа: 5, Остальные: 0).

- `chmod 755 file` — задать права (rwxr-xr-x).
    
- `chown user:group file` — сменить владельца и группу.
    
- `chgrp group file` — сменить только группу.
    

**Специальные биты:**

1. **SUID (4xxx):** Запуск программы от имени владельца (`chmod 4755 file`).
    
2. **SGID (2xxx):** Запуск от имени группы / наследование группы в папке (`chmod 2755 dir`).
    
3. **Sticky bit (1xxx):** Защита от удаления чужих файлов (`chmod 1777 /tmp`).

## 6. Окружение и Переменные

- `env` — показать текущие переменные.
    
- `export VAR="value"` — задать переменную.
    
- `echo $PATH` — список путей поиска исполняемых файлов. _(Если файл не в PATH, запускать через `./file`)._

## 7. Управление процессами

- `ps aux` — все запущенные процессы.
    
- `top` / `htop` — мониторинг процессов в реальном времени.
    
- `kill -9 <PID>` — принудительно убить процесс (SIGKILL).
    
- `jobs` — фоновые задачи текущего терминала.
    
- `bg` / `fg` — перенос задачи в фон / возврат на экран.
    

---

## 8. Сетевые утилиты

- `ip a` — сетевые интерфейсы и IP-адреса.
    
- `ping <IP>` — проверка связи.
    
- `ss -tulnp` — открытые порты (Listening) и процессы.
    
- `netstat -ano` — аналог `ss`.
    
- `nc -nvlp 4444` — запустить прослушиватель Netcat на порту 4444.

## 10. Полезные One-liners (Шпаргалка на экзамен)

**Найти SUID файлы (повышение привилегий):**

```
find / -perm -4000 2>/dev/null
```

**Найти файлы с правом записи для всех (World-writable):**


```
find / -type f -perm -0002 2>/dev/null
```

**Найти большие файлы (бэкапы, логи):**


```
find / -type f -size +100M 2>/dev/null
```

**Найти пароли в файлах конфигурации:**


```
grep -rnw '/etc/' -e 'password' 2>/dev/null
```

**Отфильтровать системные аккаунты, оставив только реальных пользователей:**


```
grep -vE "nologin|false" /etc/passwd | cut -d ":" -f1 | sort
```
